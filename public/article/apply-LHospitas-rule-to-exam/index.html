洛必达法则在高中数学导数题中的应用
2019-05-11

来看一道高考题(2010 全国新标卷):

设函数 f ( x ) = e x − 1 − x − a x 2 f(x)=e^x-1-x-ax^2 f(x)=ex−1−x−ax2
(1) 若 a = 0 a=0 a=0，求 f ( x ) f(x) f(x) 的单调区间;
(2) 当 x ⩾ 0 x \geqslant 0 x⩾0 时 f ( x ) ⩾ 0 f(x) \geqslant 0 f(x)⩾0, 求 a a a 的取值范围。

直接看第(2)问，我们分离参数得:

a ⩽ e x − x − 1 x 2 a \leqslant \frac{e^x-x-1}{x^2} a⩽x2ex−x−1​

设 g ( x ) g(x) g(x):

g ( x ) = e x − x − 1 x 2 g(x) = \frac{e^x-x-1}{x^2} g(x)=x2ex−x−1​

对 g ( x ) g(x) g(x) 求导，可得:

g ′ ( x ) = x 2 ( e x − 1 ) − 2 x ( e x − x − 1 ) x 4 g'(x)=\frac{x^2(e^x-1)-2x(e^x-x-1)}{x^4} g′(x)=x4x2(ex−1)−2x(ex−x−1)​

令分子等于 0 0 0，解得:

x = 0 x=0 x=0

但是当我们把 x = 0 x=0 x=0 代入 g ( x ) g(x) g(x) 得:

g ( 0 ) = 0 0 g(0) = \frac{0}{0} g(0)=00​

这显然不存在。但是这时可以用洛必达法则直接求出 a a a 的取值范围。根据洛必达法则:

lim ⁡ x → 0 + g ( x ) = lim ⁡ x → 0 + e x − x − 1 x 2 = lim ⁡ x → 0 + ( e x − x − 1 ) ′ ( x 2 ) ′ = lim ⁡ x → 0 + e x − 1 2 x = lim ⁡ x → 0 + e x 2 = 1 2 \begin{aligned} \lim_{x \to 0^+}g(x) & = \lim_{x \to 0^+}\frac{e^x-x-1}{x^2} \\ & = \lim_{x \to 0^+}\frac{(e^x-x-1)'}{(x^2)'} \\ & = \lim_{x \to 0^+}\frac{e^x-1}{2x} \\ & = \lim_{x \to 0^+}\frac{e^x}{2} \\ & = \frac{1}{2} \end{aligned} x→0+lim​g(x)​=x→0+lim​x2ex−x−1​=x→0+lim​(x2)′(ex−x−1)′​=x→0+lim​2xex−1​=x→0+lim​2ex​=21​​

所以

a ⩽ 1 2 a \leqslant \frac{1}{2} a⩽21​

注意：只有当 g ( x 0 ) = 0 0 g(x_0)=\frac{0}{0} g(x0​)=00​ 或 g ( x 0 ) = ∞ ∞ g(x_0)=\frac{\infin}{\infin} g(x0​)=∞∞​ 时，才能使用洛必达法则。

再来看一道题：

k x ⩾ sin ⁡ x 2 + cos ⁡ x ( x > 0 ) kx\geqslant \frac{\sin x}{2+\cos x} (x>0) kx⩾2+cosxsinx​(x>0)

分参得：

k ⩾ sin ⁡ x x ( 2 + cos ⁡ x ) k\geqslant \frac{\sin x}{x(2+\cos x)} k⩾x(2+cosx)sinx​

设 g ( x ) g(x) g(x):

g ( x ) = sin ⁡ x x ( 2 + cos ⁡ x ) g(x) = \frac{\sin x}{x(2+\cos x)} g(x)=x(2+cosx)sinx​

求导:

g ′ ( x ) = x − 2 sin ⁡ x − ( sin ⁡ x − 2 x ) cos ⁡ x x 2 ( cos ⁡ ( x ) + 2 ) 2 g'(x) = \frac{x-2 \sin x-(\sin x-2 x) \cos x}{x^2 (\cos (x)+2)^2} g′(x)=x2(cos(x)+2)2x−2sinx−(sinx−2x)cosx​

令 0 0 0, 解得:

x = 0 x = 0 x=0

代入得:

g ( 0 ) = 0 0 g(0) = \frac{0}{0} g(0)=00​

由洛必达法则:

lim ⁡ x → 0 g ( x ) = lim ⁡ x → 0 sin ⁡ x x ( 2 + cos ⁡ x ) = cos ⁡ x − x sin ⁡ x + cos ⁡ x + 2 = 1 3 \begin{aligned} \lim_{x \to 0} g(x) & = \lim_{x \to 0} \frac{\sin x}{x(2+\cos x)} \\ & = \frac{\cos x}{-x \sin x+\cos x+2} \\ & = \frac{1}{3} \end{aligned} x→0lim​g(x)​=x→0lim​x(2+cosx)sinx​=−xsinx+cosx+2cosx​=31​​

故

k ⩽ 1 3 k \leqslant \frac{1}{3} k⩽31​

以上。
